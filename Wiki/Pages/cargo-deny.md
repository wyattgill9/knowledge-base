# cargo-deny

cargo-deny audits your Rust dependency graph for license violations, known vulnerabilities, duplicate versions, and banned crates. Run it in CI with the EmbarkStudios GitHub Action. Complements cargo-audit (RustSec scanning) and cargo-vet (formal audit trails). Part of the recommended [[rust-build-tooling]] stack.
