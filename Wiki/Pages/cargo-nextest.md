# cargo-nextest

cargo-nextest is a parallel test runner for Rust that replaces `cargo test`. It runs each test in its own process, provides better output formatting, and supports retries, filtering, and JUnit XML output for CI. Part of the recommended [[rust-build-tooling]] stack.
