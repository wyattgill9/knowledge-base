# criterion

criterion is Rust's mature statistical benchmarking framework with HTML reports and detailed statistical analysis. It remains functional but has seen reduced maintainer activity. For new projects, [[divan]] is the recommended alternative with a more ergonomic API and active development. See [[iai-callgrind]] for deterministic CI benchmarks.
